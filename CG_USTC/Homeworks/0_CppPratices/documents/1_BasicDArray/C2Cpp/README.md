# C to C++

此为 [1_BasicDArray](../) 和 [3_TemplateDArray](../../3_TemplateDArray/) 中的相关项目

包含七个子项目

- Array：固定长度数组
- Array0：动态数组功能实现
- Array 1.0：用 `struct` 打包 `p` 和 `n`，引用形式
- Array 1.1：用 `struct` 打包 `p` 和 `n`，指针形式
- Array 2：函数放到 `struct` 中
- Array 3：改用 `class` 
- Array 4：头文件和实现文件
- Array 5：模板 `template`
